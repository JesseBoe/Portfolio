My work as a Learning Assistant with Trilogy Education.

Currently, I work with Trilogy Education to support countless curious students tackle difficult web development questions. When students are stuck and are unable to proceed with their course work, that's where I come in and provide a path forward to the student. Often times this is done through a Zoom call where I can explain to the student, the reason why they are getting a specific error or unexpected behavior. Through the use of online examples and resources I will help re-enforce the learning material and leave the student with the tools and knowledge to fix the issues they were encountering. I enjoy my work a lot since it gives me an opportunity to make an impact with so many students different students.

Students are able to leave feedback for each Learning Assistant they work with, and these metrics are provided to us. I take great pride my work with my average student rating of 4.9 / 5.0 , a faster than average solve time, as well as a higher feedback ratio from the students I work with.

Recent projects

Say What?

"Say What?" is a browser game built using the MERN (Mongo Express React Node.js) stack. It plays much like wheel of fortune, but with phrases from pop culture. It contains many features such as user authentication, websockets, stat tracking and an in game chat!

Dino Must Die

Dino Must Die is a 2D platformer I built with the Unity game engine using C#. It features many collectable power-ups, enemies and tricky puzzles! It even has online multiplayer via a websocket connection and a Node.js server so you can play with your friends!

One Step From Modding

One Step From Modding is a front end website I built using React with the goal of teaching people how to add content to a game called One Step From Eden. Its regarded as the best place for beginers to learn how to use lua, C# and xml to make their own creations on the steam workshop.
