Recent projects

Say What?

"Say What?" is a browser game built using the MERN (Mongo Express React Node.js) stack. It plays much like wheel of fortune, but with phrases from pop culture. It contains many features such as user authentication, websockets, stat tracking and an in game chat!

Dino Must Die

Dino Must Die is a 2D platformer I built with the Unity game engine using C#. It features many collectable power-ups, enemies and tricky puzzles! It even has online multiplayer via a websocket connection and a Node.js server so you can play with your friends!

Open Job Pipeline

Open Job Pipeline is a HR tool I built using React. It currently is purely a front-end app, but it features a custom built data-table that the user can sort and search. It has all CRUD options, pagination and a login / signup page.

One Step From Modding

One Step From Modding is a website I built using React with the goal of teaching people how to add content to a game called One Step From Eden. Its regarded as the best place for beginers to learn such a thing.

Randomizer Catalog

Randomizer Catalog is an app I built using the SERN stack. A randomizer is a tool that takes a game and randomly changes elements to create a whole new experience. I made a website to make locating such tools easier.
