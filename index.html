Dino Must Die

Dino Must Die is a 2D platformer built with the Unity game engine. It features custom physics using raycasts and vector maths giving the player much more precision when compared to Unity's stock controller. It also has collectable power-ups, enemies and tricky puzzles!

Invite your friends to play along with you. The game can connect to a Node.js server through a Websocket, allowing you to even play online!

The game can get pretty challenging, so be prepared! This game was solely developed by me, however some of the pixel art comes from free online resources. That being said, I owe a huge thanks to all the great tutorials out there on the internet!

Say What?

Say what? is a browser game built using the MERN (Mongo Express React Node) stack. It plays much like wheel of fortune, but with popular phrases from pop culture. It features user authentication, websockets, stat tracking and an in game chat! We made this for our last group project at the UW bootcamp.

Any action a user takes such as guessing a letter, spinning the wheel, or attempting to solve the puzzle ends up being verified by the server. It was for this reason that it was important that we kept most of the logic off of the client side.

Websockets can be used to keep our game real time. Normally when using just http, you would have to use a technique called polling, where the client asks the server if there is any new information for them at a set time interval. With sockets you create a two way connection where at any time both client and server can send data through. Many popular websites such as Twitter and Gmail are using websockets.

For authentication we used Passport.js. Passport allows you to quickly implement many different ways for the user to log in. They call these different ways 'strategies'. We decided to use both a local strategy and the Google-oauth strategy. This is quite nice because going through the trouble of making a new account can be a bit tedious, but signing in with Google is quite easy.
